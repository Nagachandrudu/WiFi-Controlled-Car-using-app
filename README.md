# ESP32 WiFi Controlled Car

This project is a WiFi controlled robotic car built using ESP32 and L298N motor driver.

The car can be controlled through a web browser using WiFi network.

## Features
• Wireless control using WiFi
• Real time movement control
• Forward, backward, left and right movement
• Simple web interface

## Hardware Components
• ESP32
• L298N Motor Driver
• DC Motors
• Robot Chassis
• Power Supply

## Software
• Arduino IDE
• Embedded C

## Working Principle
ESP32 creates a WiFi server. 
When a user connects to the IP address of ESP32 through a browser, control buttons appear on the webpage.

When a button is pressed, ESP32 receives the command and sends signals to the L298N motor driver to move the car.

## Applications
• Robotics learning
• IoT based vehicle control
• Wireless robotic systems
